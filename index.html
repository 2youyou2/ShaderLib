The Evil Tentacles

A game entry for Ludum Dare 29 by Wu-Hao

Based on Cocos2d-js 3.0 and Chipmunk js

you play as the Kraken, wreak havoc on the sea and eat surviving sailors. The game is 100% physics simulated and can be laggy on legacy browser or old computers.

Only tested on Chrome and Firefox, for best performance, please use Chrome

The game isn't that finished, it lacks winning and losing, but it is quite fun as is, as I spent most of the time fiddling with physics

If you like this game, and want to see a more complete version, please vote me on Ludum Dare
